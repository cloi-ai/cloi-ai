cloi runs entirely on your machine, parses your tracebacks, and generates structured patches with an on‑device LLM—no API keys or cloud costs.

$
npm install -g @cloi-ai/cloi

On‑device LLM

Cloi uses Ollama to run models locally so your data never leaves.

Context‑aware fixes

One‑command patching

Approve and apply patches without leaving the terminal.

Get started in seconds

1. Install globally

$
npm install -g @cloi-ai/cloi

No API key needed, runs completely locally via Ollama.

2. Navigate to your project

$
cd your-project-directory

3. Debug errors instantly

$
cloi

Run cloi when you encounter errors to generate fixes for your code.
